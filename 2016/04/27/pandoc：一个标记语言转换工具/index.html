pandoc：一个标记语言转换工具

发表于 2016-04-27   |  

转载自：Phodal

Pandoc是一个标记语言转换工具，可实现不同标记语言间的格式转换，堪称该领域中的“瑞士军刀”。

可以将 markdown、 reStructuredText、 textile、 HTML、 DocBook、 LaTeX、 MediaWiki markup、 TWiki markup、 OPML、 Emacs Org-Mode、 Txt2Tags、 Microsoft Word docx、 LibreOffice ODT、 EPUB、 Haddock markup

转化为

XHTML、 HTML5、 以及 HTML幻灯片 (Slidy / S5等)，或者DZSlides、Microsoft Word docx、 OpenOffice/LibreOffice ODT、 OpenDocument XML、EPUB、DocBook、 GNU TexInfo、 Groff man pages、LaTeX、 ConTeXt、 LaTeX Beamer slides、PDF via LaTeX、Markdown、 reStructuredText、 AsciiDoc、 MediaWiki markup、 Emacs Org-Mode、 Textile

我最常用的就是：将md转化为workd及pdf(如果要转为PDF，则需要另外的一个工具——LaTeX)。我的毕业论文及之前的几本电子书都是这么做的，
它是一个命令行工具，安装方式如下：

  • Windows: choco install pandoc
  • Ubuntu/CentOS/OpenSUSE: apt-get intall pandoc 或者 yum install pandoc
  • Mac OS: brew install pandoc

个人感觉比较常用的几种转换命令：

  • HTML fragment: pandoc README(原文件) -o example1.html(希望输出为何种格式文件)
  • Standalone(独立的) HTML file(这个感觉很有用额): pandoc -s README(原文件) -o example2.html
  • HTML slide shows: pandoc -s --webtex -i -t slidy(你使用的幻灯片框架名称) SLIDES(原文件) -o example3.html
  • Syntax highlighting(语法高亮) of delimited code blocks: pandoc code.text -s --highlight-style pygments(其他可选值：kate /monochrome /espresso /haddock /tango /zenburn) -o example18a.html
  • EPUB ebook: pandoc -S README -o README.epub
  • Word docx: pandoc -s -S README -o example29.docx

虽然可以转换，但是效果就不尽如人意了(例如：排版、语法高亮、转html可能乱码等等)
